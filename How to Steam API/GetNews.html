Get News for App Call

The GetNewsForApp call is used for getting news articles posted about or for a game. The call returns information on the article in the format desired.

This Call requires four arguements and results in the appid and the newsitems(array of news info)

To start lets look at the four arguements needed to make the call.

Inputs

  • appid- The ID of the game
  • count- Amounts of Articles to be displayed
  • maxlength-How much of the article you want to read before it ...(usually 300ish)
  • format- what format you want it displayed in (JSON,XML,VDF)

The Call

So now lets make the call!

http://api.steampowered.com/ISteamNews/GetNewsForApp/v0002/?appid=230410&count=3&maxlength=300&format=json

This uses values appid=230410(warframe),count=3(articles,maxlength=300(characters),format=json

In the image below we can see a sample of how this looks with these parameters.

The Output from this shows many objects inside the displayed array. Each object has a meaning to it. Here is a decription for each of the objects

  • appid--displays the game id
  • gid-- displays the post id
  • title -- displays the article title
  • url-- link to article page
  • is_external_url-- says if the url is an external page(good to check in case we don't want to trust page)
  • Author--Author of the article
  • contents--The Article context limited by maxlength
  • feedlable-- Name of where the article comes from, i.e. website, announcements, update,etc.
  • Date--Date published, displayed in unix time so make sure to convert
  • Feedname--Feedname is a more parserable(readable/computer friendly) feedlable, it allows us to parse it and get all the updates, everything from pc gamer, announcements, etc.

To recap the call needs 4 inputs: appid, count, maxlength, and format. Don't Forget to also put the v0002 after GetNewsForApp/ its part of the call and can trip you up.
