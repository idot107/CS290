What are those??

Some achievements are hard, some are easy, but first you will want to know what they are!

The GetPlayerAchievements call uses three arguements to give you details about a games achievements and player completion.

Inputs

  • appid--The id of the game you want to look at
  • key-- your api key that you got on the api key page
  • steamid=the id of your steam id or a players steam id.

The Call

http://http://api.steampowered.com/ISteamUserStats/GetPlayerAchievements/v0001/?appid=48700&key=XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX&steamid=76561197972495328

The key has to be your own api key(or someone elses, but i cant endorse stealing) This will give you an output looking something like this:

The data listed has four outputs:

  • steamID-- the user id that is accessing
  • gameName--The Game name... Simply states the game
  • apiname--the name of the achievement in steams naming system
  • achieved--1= achieved 0=not achieved

There seems to be missing data on some games so you may want to be careful with how you set up your parsing and make sure it has a no information option.
