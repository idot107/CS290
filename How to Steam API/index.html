How to use Steam's API


Guide Basics

Welcome to the How-To guide for Steams Web API

The Steam web API can be used to access data about steam users and games. It is done through either JSON or XML objects returned from different queries.

This guide will give examples of the different queries that are avalible to use on the steam API and show you why they are important.

This how to guide will cover the following JSON calls:

  1. GetNewsForApp
  2. GetGlobalAchievementPercentageForApp
  3. GetPlayerSummaries
  4. GetUserStatsForGame
  5. GetFriendList
