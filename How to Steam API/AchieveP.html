How valuable or rare is each achievement?

The steam API allows us to see the global percentage of achievements, well achieved.

To begin lets look at the arguements that have to be passed in:

Inputs

  • gameid--The id of the game
  • format--format(JSON,XML,VDF)

This takes only two inputs but has a whole lot of outputs so be ready!

The Call

http://api.steampowered.com/ISteamUserStats/GetGlobalAchievementPercentagesForApp/v0002/?gameid=15620&format=VDF

The output looks something like this!

Just as a heads up this is VDF format! You may notice that there is a number before each achievement information! This isn't in either XML or JSON forms and may be of use if you want to count up all the achievements without a program to parse and then determine.

There are two outputs per achievement:

  1. name--the descriptive name of the achievement(not what is displayed on steams UI)
  2. percentage--number already in %, the result is from the formula (#owns game/#has achievement)x100

The list ends up being very long, since most games have upwards of 40 to 200 achievements. All of which dont use steams achievement names, making it very inconvienent for use. Luckily this can be used in combination with GetPlayerAchievements which we learned in the previous page how to use.
